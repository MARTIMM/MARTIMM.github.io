MongoDB and BSON Howto

The perl 6 modules MongoDB and BSON are maintained by me and can be found here and here. However, the modules are also registered in the perl 6 module library so when panda is installed you can also install them using panda;

> panda install MongoDB
> panda install BSON
The last command is not realy necessary because the MongoDB is dependend on it and panda will install it automatically.

The minimal use of the MongoDB driver is loading the driver, make contact with the server, then do your thing and close it down.

use MongoDB;

my MongoDB::Connection $connection .= new( :host('localhost'), :port(27017));
my MongoDB::Database $database = $connection.database( 'test');
my MongoDB::Collection $collection = $database.collection( 'perl_users');
