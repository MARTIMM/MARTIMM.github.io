Perl 6 Crossroads

  • perl documentation
    An important site for the Perl 6 student. You can find information about every possible section of the language ranging from classes and objects, containers etc to grammars and regular expressions. There is also good information about the many types used in Perl 6 and built-in routines under which there are operators (Almost everything in Perl are objects and operators are functions or methods).

  • perl 6 planet
    Latest news(weekly).about changes and bug fixes in and around perl 6. Many pointers to blogs with examples and a little rant here and there about some service.

  • perl6 smoke tests
    A list of test results is shown where rakudo is changed to a new version. I am not doing well at the moment :-/. See MARTIMM.

  • perl6 module test reports
    A list of test results of Perl6 modules based on reports send by panda when the environment variable PANDA_SUBMIT_TESTREPORTS is set to 1. These reports are more detailed in that it makes the operating systems visible on which it is installed. It is also visible which compiler version its backend is used and which module version of the module is tested.
